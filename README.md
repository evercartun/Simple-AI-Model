# Simple-AI-Model
This is a simple tensorflow ai model that I build. CNN and image recognition implimentation.

It is only a matter of time before we adapt Artificial Intelligence fully. 
Throught AI, that has been changes in the development sector in the introduction of a new concept called ##vibe-coding

This is an illustration of code, built using Python's modules called tensorflow. With a small neural network enough to comprehend images.
Hence the CNN intergration. Open source project. All you have to do is train it and save the trained data. After that you are able to use it yourself. From here anyoone can increase or decrease the code.

To avoid crashing and error displays, it is adviced to install the following dependancies into your system:
.Tensorflow
.Scikit-learn
.pandas
.numpy

In your terminal whether windows, mac or linux. The installation of dependancies is issued with a single command
"pip install <name-of-dependancy>"

Note: "pip install Tensorflow" automatically installs all the necessary dependancies such as pandas.

Note: It is adviced that you use the python version 3.10. in linux just type "apt install python3.10" and in mac and windows install from the official python website.
this is because, the tensorflow dependancy is not included in new versions beyond 3.10
